## 🚀 新的启动方式

### 方法1：进入m-server目录启动
```bash
# 1. 进入项目目录
cd m-server

# 2. 安装依赖（首次运行）
pip install -r requirements.txt

# 3. 启动服务
python run.py
```

### 方法2：从test1根目录启动
```bash
# 直接从test1目录启动m-server中的服务
python m-server/run.py
```
